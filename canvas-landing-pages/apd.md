## About

In this lab, you will focus on achieving 70-80% test coverage for your API using Pytest. This ensures that the core functionality of your application is well-tested without getting bogged down in unlikely edge cases. Prioritizing essential features makes tests easier to maintain as your code evolves, reducing long-term costs and keeping your test suite relevant. By the end of this lab, you will have a deeper understanding of test coverage and the ability to implement effective unit tests using Pytest.
